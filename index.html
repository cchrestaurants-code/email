🔑 Settings Password
Protect Settings & Admin with a password. Staff who open this file cannot access Settings without the correct password — even if they clear their browser.
How it works:
The password hash is stored inside this file, not in the browser. This means clearing the browser cannot bypass it.

When you set a password, you'll get a hash code. Send it to Shazwan who will update the file with that hash — making it permanent across all devices and browsers.
